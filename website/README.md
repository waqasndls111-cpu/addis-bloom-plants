# Oral Surgery Static Website Template

This VS Code-ready project is a static HTML/CSS/JavaScript build inspired by a premium oral surgery practice website.

## What changed in V2

- Added reference-style top navigation arrows.
- Added desktop hover dropdowns for Our Center, Services, Patient Center, and Contact Us.
- Added nested right-side flyout menus for Our Doctors, Bone Grafting, Oral & Maxillofacial Surgeries, and Facial Birth Deformities.
- Added mobile click-to-open dropdown and nested submenu behavior.
- Added 39 starter interior pages so menu items link to actual `.html` files instead of broken URLs.

## File structure

```text
index.html
about-our-office.html
our-doctors.html
... additional menu pages ...
css/styles.css
js/script.js
assets/
```

## Run locally

Open `index.html` in your browser or use the VS Code Live Server extension.

## Replace before publishing

The design uses placeholder SVG images, neutral practice branding, and starter copy. Replace the practice name, phone number, addresses, credentials, reviews, medical content, forms, maps, and imagery with your own approved content and licensed assets.
