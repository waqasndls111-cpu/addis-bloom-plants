## Locator strategy

Avoid generated or dynamic SmartMenus IDs like these:

```text
sm-17784571195898584-7
sm-1778457410377533-7
sm-17784574103800159-7
sm-17784585880032534-7
sm-17784585880074558-7
```

Use stable selectors instead:

```text
//header//nav[contains(@class,'elementor-nav-menu--main') and not(ancestor::*[contains(@class,'elementor-sticky__spacer')])]
//a[normalize-space()='Services']
//a[normalize-space()='Bone Grafting' and contains(@href,'/bone-grafting/')]
//a[normalize-space()='Bone Grafting FAQs' and contains(@href,'/bone-grafting-faqs/')]
//a[normalize-space()='Ridge Augmentation' and contains(@href,'/ridge-augmentation/')]
//a[normalize-space()='Sinus Lift' and contains(@href,'/sinus-lift/')]
//a[normalize-space()='Platelet Rich Fibrin (PRF)' and contains(@href,'/platelet-rich-fibrin-prf/')]
```

## Duplicate nav warning

Elementor renders multiple menu copies:

- desktop header nav
- mobile dropdown nav
- sticky spacer nav copy

The automation targets only the visible desktop header nav and excludes any ancestor with class `elementor-sticky__spacer`.
